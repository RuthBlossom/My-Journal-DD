# Blog App

## Description

The Blog App is a web application built using Node.js and MongoDB that allows users to create, view, and manage blog posts. It utilizes Express.js for handling routes, EJS for rendering views, and Mongoose for interacting with MongoDB.

## Features

- **Home Page**: Displays a list of all blog posts with their titles and excerpts.
- **Individual Post Pages**: Allows users to view the full content of each blog post by clicking on its title.
- **Compose Page**: Provides a form for users to create new blog posts with a title and content.
- **About Page**: Offers information about the app and its creator.
- **Contact Page**: Allows users to get in touch with the app creator.

## Technologies Used

- **Node.js**: JavaScript runtime for building server-side applications.
- **Express.js**: Web application framework for Node.js used for routing and middleware.
- **MongoDB**: NoSQL database used for storing blog post data.
- **Mongoose**: MongoDB object modeling tool for Node.js used for interacting with MongoDB.
- **EJS**: Embedded JavaScript templating language for generating HTML with JavaScript.

## Installation

1. Clone the repository to your local machine.
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory.
   ```
   cd <project-directory>
   ```
3. Install dependencies using npm.
   ```
   npm install
   ```
4. Start the server.
   ```
   node app.js
   ```
5. Access the application in your web browser at `http://localhost:3000`.

## Usage

- Visit the home page to view existing blog posts or create new ones.
- Click on a blog post title to view its full content.
- Use the navigation links to access the about and contact pages.
